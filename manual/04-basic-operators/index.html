Swift 3.0 语法速查 基本运算符
1. 赋值运算符: = 
let b = 10
var a = 5
a = b                                  // a为10,"="的意思:把b的值赋值给a
2. 多元组:同时定义多个常量或者变量 
let (x, y) = (1, 4)                    // 多元组会被自动分解多个常量变量
print(x)                               // 1, x为常量
print(y)                               // 4, y为常量
3. 算术运算符: + - * / 
1 + 1                                  // 2
3 - 1                                  // 2
1 * 2                                  // 2
6 / 3                                  // 2
print("Hello"+"World")                 // HelloWorld, 利用"+"可以进行字符串的拼接
4. 求余运算符: % 
7 % 3                                  // 1, 2*3+1
-11 % 3                                // -2, 11前的负号能被保留
11 % -3                                // 2, 3的负号会被忽略,与上面-11 % 3的取余不一样
5. 组合赋值运算符: += 
a += 2                                 // 12, 等价于 a = a + 2
6. 比较运算符: == != > < >= <= 
3 == 4                                 // false
3 != 4                                 // true
3 > 4                                  // false
3 < 4                                  // true
3 >= 4                                 // false
3 <= 4                                 // true
7. 三目运算符: a ? b : c 
let c = a == 2 ? 7 : 8                 // 8, 如果a==2为true,那么c为7,否则为8
8. 空合运算符: a ?? b 
let optionalE: Int? = 4                // ?? 前面那个值要为可选类型
let d = optionalE ?? b                 // 4, 如果optionalE为空,d为b的值
                                       // 如果不为空就是optionalE解包后的值
9. 区间运算符: 表示一个区间的值 
1...5                                  // 定义一个1到5的数,包含1和5
1..<5                                  // 定义一个1到5的数,只包含1
10. 逻辑运算符: 逻辑非:!a 逻辑与:a&&b 逻辑或:a||b 
!true                                  // false
true && true                           // true
false || false                         // false
